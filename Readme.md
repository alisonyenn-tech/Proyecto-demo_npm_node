# Proyecto básico Node.js con NPM

Este proyecto es un ejemplo básico de un servidor en Node.js utilizando NPM y Express.

## Requisitos
- Node.js
- NPM

## Instalación
1. descargar el proyecto
2. Abrir la carpeta del proyecto
3. Ejecutar el comando:

npm install

## Ejecución del proyecto

### Modo normal
npm run start

### Modo desarrollo (nodemon)
npm run dev

## Ruta disponible
- `/` → Muestra el mensaje "Hola, Node.js con NPM"
